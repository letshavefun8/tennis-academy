# Академия тенниса

Игровой тренажёр по тактике тенниса с маскотом Эйсом 🎾. 116 вопросов в 17 блоках, очки-мячи, звания, «Тренировка дня» и вторая подача.

## Как открыть

Откройте файл `index.html` в браузере — двойным кликом или командой:

```
open index.html
```

Интернет не нужен. Работает полностью офлайн по `file://`.

## Механика

### Гейм
- Каждый блок или «Тренировка дня» — гейм из 7 вопросов (меньше, если в блоке меньше).
- Отбор умный: сначала незнакомые (корзина A) → ранее ошибочные (B) → уже освоенные (C).
- Лёгкий старт: первыми идут до 2 вопросов базового (Б) уровня.
- Сортировка внутри гейма: Б → С → П.

### Вторая подача
- Неверный ответ с первой попытки → вопрос добавляется в конец очереди (вторая подача).
- Верный со второй — начисляется половина очков; точка становится зелёной.
- Неверный со второй — очков нет, точка остаётся жёлтой, вопрос попадает в «Разобрать ошибки».

### Очки 🎾
| Сложность | 1-я подача | 2-я подача |
|-----------|-----------|-----------|
| Б (базовый) | 10 | 5 |
| С (средний) | 20 | 10 |
| П (продвинутый) | 30 | 15 |

- **Серия**: при 3 верных подряд включается множитель ×2 (с 4-го ответа).
- Серия сбрасывается при ошибке; очки при этом не сгорают.

### Звания
| Порог | Звание | Эмодзи |
|-------|--------|--------|
| 0     | Новичок | 🎒 |
| 300   | Юниор | 🥎 |
| 800   | Игрок основы | 🎾 |
| 1600  | Капитан команды | 🏅 |
| 2800  | Мастер корта | 🏆 |
| 4500  | Чемпион Академии | 👑 |

### Звёзды блока
- ★ — отвечено верно хотя бы раз ≥ 50% вопросов блока.
- ★★ — ≥ 80%.
- ★★★ — 100%.

## localStorage: схема профиля

Ключ: `tennisAcademy.v1`

```json
{
  "v": 1,
  "points": 420,
  "onboarded": true,
  "history": {
    "1": { "c": 2, "w": 0, "last": 1, "t": 1718000000000 },
    "5": { "c": 0, "w": 1, "last": 0, "t": 1718001000000 }
  }
}
```

- `points` — общий счёт (целое ≥ 0).
- `onboarded` — показан ли онбординг (boolean).
- `history` — история по каждому вопросу (ключ = строка `q.id`):
  - `c` — количество верных ответов.
  - `w` — количество неверных ответов.
  - `last` — результат последней попытки: 1 = верно, 0 = неверно.
  - `t` — `Date.now()` последнего ответа.

## Как сбросить прогресс

Откройте DevTools браузера → вкладка **Application** → **Local Storage** → удалите ключ `tennisAcademy.v1`. После перезагрузки страницы начнётся онбординг.

Или в консоли:
```js
localStorage.removeItem('tennisAcademy.v1');
location.reload();
```

## Как перегенерировать данные

Если вы изменили `tennis_question_bank.md`, пересоздайте `questions.js`:

```
/usr/local/bin/python3.13 convert.py
```

Скрипт выведет `OK: 116 вопросов, 17 блоков` и обновит файл `questions.js`.

## Структура проекта

- `tennis_question_bank.md` — исходный банк вопросов (не редактировать)
- `convert.py` — конвертер из markdown в JS
- `questions.js` — сгенерированные данные (не редактировать)
- `index.html` — разметка (4 экрана)
- `style.css` — игровые стили
- `app.js` — логика Академии
